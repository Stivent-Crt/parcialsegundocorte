# Sistema de Gestión de Biblioteca

## Descripción
Sistema orientado a objetos para administrar una biblioteca universitaria.

## Características
- Gestión de usuarios y bibliotecarios
- Gestión de libros
- Préstamo y devolución de libros
- Validación de datos (edad > 0, códigos numéricos)

## Requisitos
- Compilador C++ (g++ recomendado)

## Compilación
```bash
g++ -o biblioteca main.cpp src/*.cpp -I include
./biblioteca
